Introduction to Amazigh Script

The Amazigh language (also known as Berber) is written in the Tifinagh alphabet. This ancient script has been modernized and standardized as Neo-Tifinagh for use across North Africa, particularly in Morocco, Algeria, and other regions with Amazigh populations.

This website will help you learn the Tifinagh alphabet through interactive lessons and practice exercises.
